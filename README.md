# Proyecto Reciclador Inteligente 
# Autores: Diego Casanova - Jordy Velasquez

Código fuente del proyecto para un sistema clasificador de reciclaje inteligente usando ESP32 y sensores inductivos y de color.
Este repositorio contiene el código fuente del sistema de reciclaje inteligente.

## Descripción
Sistema basado en ESP32 para clasificación automática de residuos.

## Código
El código principal del microcontrolador se encuentra en el archivo 'codigo_proyecto'.
